# 🔥 Flask CRUD Application with MySQL

A **simple Flask web application** that performs **CRUD operations** (**Create, Read, Update, Delete**) using **MySQL** as the database backend.

---

## 📌 Features

* ➕ **Create** new records
* 📖 **Read** existing records
* ✏️ **Update** records
* ❌ **Delete** records
* 🗄️ **MySQL database integration**

---

## 🛠️ Built With

* 🐍 **Flask (Python)**
* 🌐 **HTML, CSS, Bootstrap**
* 🗄️ **MySQL**

---

## ⚡ Description

This project demonstrates how to **connect a Flask web application to a MySQL database**.
It’s perfect for beginners learning **Flask CRUD operations** and **relational database integration**.
